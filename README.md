Interactive graphs used to update an online book: A synthesis of scientific research at Hubbard Brook. 
These graphs read data directly from the Environmental Data Initiative repository and are rendered with 
plotly htmlwidgets for display in the Hubbard Brook Research Synthesis book that is available online at 
[hubbardbrook.org](https://hubbardbrook.org). In the main repo you will find the code to create these 
graphs and you will find the rendered graphs here: [https://hbr-lter.github.io/OnlineBookGraphs/](https://hbr-lter.github.io/OnlineBookGraphs/). 
This work has been supported by students in the Environmental Data Science Capstone at Virginia Tech.
